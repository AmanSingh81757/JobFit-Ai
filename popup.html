How to get an API Key?

Generating Your Gemini API Key

Follow these steps to get your API key from Google AI Studio:

  1. Go to Google AI Studio.
  2. Sign in with your Google account.
  3. Click on "Get API key in a new project" or "Create API key" if you have an existing project.
  4. Copy the generated API key.
  5. Paste the API key into the field above and click "Save Key".
